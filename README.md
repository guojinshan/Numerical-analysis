# 数值分析课程

包含线性代数基础；浮点数计算；直接法求解线性系统（LU分解，Gauss消除，Cholesky—）以及系统稳定性的判断；最小二乘法与标准方程；迭代法求解线性系统（Jacobie和Gauss-Seidel方法）和非线性函数的零点（二分法，定点法，牛顿法，截距法）；插值法（牛顿, 拉格朗日, 三次样条）；数值积分；微分方程；特征值和特征向量等内容，以及基于Scilab的相关算法的代码实现。
